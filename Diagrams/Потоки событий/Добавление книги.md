# Поток событий для прецедента "Добавление книги":



### Основной поток:

1. Библиотекарь выбирает пункт меню, чтобы добавить запись разных книг. 
2. Библиотекарь вводит данные о книге. 
3. Запись книг и присвоение номера каждой книге.


### Альтернативный поток событий:
1. Если логин введен неверно Библиотекарь должен  повторить попытку.
